# Real-Time AI-Powered Network Firewall

This project implements a real-time, deep learning-based intrusion detection system (IDS) utilizing a Convolutional Neural Network and Long Short-Term Memory (CNN-LSTM) architecture. Trained on the CICIDS2017 dataset, the model analyzes bidirectional network flows to distinguish between benign traffic and malicious attacks with high accuracy.

This repository/guide covers the deployment and real-time testing of the trained model on Kali Linux.

## 📋 Prerequisites

Before starting, ensure you have the following on your Kali Linux machine:
1. **Kali Linux** (Physical machine or VM) with root access.
2. The three exported artifacts from the training pipeline:
   * `model.keras` (The trained CNN-LSTM model)
   * `scaler.pkl` (The fitted StandardScaler)
   * `features.pkl` (The exact list of 77 numeric features)

Python
# realtime_firewall.py
import time
import pandas as pd
import numpy as np
import joblib
import pickle
from tensorflow.keras.models import load_model

print("[*] Loading AI Firewall Model and Scaler...")
model = load_model('model.keras')
scaler = joblib.load('scaler.pkl')

with open('features.pkl', 'rb') as f:
    expected_cols = pickle.load(f)

LIVE_CSV = "live_traffic.csv"  
processed_rows = 0

print("[*] Monitoring network flows in real-time...")

while True:
    try:
        df = pd.read_csv(LIVE_CSV)
        
        if len(df) > processed_rows:
            new_flows = df.iloc[processed_rows:].copy()
            processed_rows = len(df)
            
            # Clean data
            new_flows.columns = new_flows.columns.str.strip().str.replace(' ', '_')
            new_flows.replace([np.inf, -np.inf], np.nan, inplace=True)
            new_flows.fillna(0, inplace=True) 
            
            # Extract features
            try:
                X_live = new_flows[expected_cols].astype(np.float32)
            except KeyError:
                continue 
                
            # Scale and Reshape
            X_live_scaled = scaler.transform(X_live)
            X_live_reshaped = X_live_scaled.reshape((X_live_scaled.shape[0], 1, X_live_scaled.shape[1]))
            
            # Predict
            predictions = model.predict(X_live_reshaped, verbose=0)
            
            # Alert
            for i, pred in enumerate(predictions):
                src_ip = new_flows.iloc[i].get('src_ip', 'Unknown IP') 
                if pred[0] > 0.5:
                    print(f"[!] ALERT: Malicious Flow from {src_ip} | Confidence: {pred[0]*100:.2f}%")
                else:
                    print(f"[+] Benign flow from {src_ip} | Confidence: {(1-pred[0])*100:.2f}%")
                    
    except (FileNotFoundError, pd.errors.EmptyDataError):
        pass 
    
    time.sleep(2) 
🚀 Phase 3: Real-Time Execution
To run the firewall, you will need three separate terminal windows. Ensure you know your network interface name (e.g., eth0 or wlan0) by running ip a.

Terminal 1: The Sniffer
Extracts features from raw packets and writes them to a CSV.

Bash
sudo su
cd ~/Desktop/AIFirewall
source venv/bin/activate
cicflowmeter -i eth0 -c live_traffic.csv
(Note: Replace eth0 with your active interface).

Terminal 2: The AI Engine
Reads the CSV, processes the data, and outputs predictions.

Bash
sudo su
cd ~/Desktop/AIFirewall
source venv/bin/activate
python realtime_firewall.py
Terminal 3: The Attacker (Testing)
Simulate attacks against your own machine to trigger the AI detection. Use ifconfig or ip a to find your Kali IP address (e.g., 192.168.1.X).

Test 1: Aggressive Port Scan (Nmap)

Bash
nmap -A -T4 <your_kali_ip>
Test 2: SYN Flood / DoS (hping3)

Bash
sudo hping3 -S -p 80 --flood <your_kali_ip>
Watch Terminal 2. As the attack traffic is processed, the AI model will identify the malicious patterns and print the [!] ALERT warnings to the screen.
